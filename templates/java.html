Tools and Setup

To get started with Java, you'll need the following tools:

  • Java Development Kit (JDK): Download and install the latest JDK from Oracle or use Eclipse Adoptium.
  • Integrated Development Environment (IDE): Recommended IDEs include:
    • IntelliJ IDEA: A powerful IDE for Java development.
    • Eclipse: A free, open-source IDE widely used in the industry.
    • VS Code: Lightweight with excellent Java extensions.

After installing the JDK, set up your JAVA_HOME environment variable and add the bin directory to your system's PATH to enable running Java commands in the terminal.

Java Basics

Here are some basic concepts and examples to get you started:

Variables

Variables are used to store data. Here's an example:


public class VariablesExample {
    public static void main(String[] args) {
        int number = 42; // Integer
        double price = 19.99; // Decimal
        String message = "Hello, Java!"; // Text
        System.out.println("Number: " + number);
        System.out.println("Price: $" + price);
        System.out.println("Message: " + message);
    }
}

Control Flow

Control flow statements like if, else, and loops are fundamental in Java:


public class ControlFlowExample {
    public static void main(String[] args) {
        int number = 10;

        // If-Else Statement
        if (number > 5) {
            System.out.println("Number is greater than 5");
        } else {
            System.out.println("Number is 5 or less");
        }

        // For Loop
        for (int i = 0; i < 5; i++) {
            System.out.println("Count: " + i);
        }
    }
}

Object-Oriented Programming

Java is an object-oriented programming (OOP) language. Here's an example of creating a simple class with a method:


class Car {
    String brand;
    int speed;

    // Constructor
    Car(String brand, int speed) {
        this.brand = brand;
        this.speed = speed;
    }

    // Method
    void displayInfo() {
        System.out.println("Brand: " + brand + ", Speed: " + speed + " km/h");
    }
}

public class OOPExample {
    public static void main(String[] args) {
        Car car = new Car("Toyota", 120);
        car.displayInfo();
    }
}

Working with Files

Java makes it easy to read and write files. Here's an example of writing to a file:


import java.io.FileWriter;
import java.io.IOException;

public class FileExample {
    public static void main(String[] args) {
        try {
            FileWriter writer = new FileWriter("example.txt");
            writer.write("Hello, File!");
            writer.close();
            System.out.println("File written successfully!");
        } catch (IOException e) {
            System.out.println("An error occurred: " + e.getMessage());
        }
    }
}

Connecting to a Database

Java provides robust support for database interaction using JDBC. Here's an example of connecting to a PostgreSQL database:


import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class DatabaseExample {
    public static void main(String[] args) {
        String url = "jdbc:postgresql://localhost:5432/mydatabase";
        String user = "username";
        String password = "password";

        try (Connection conn = DriverManager.getConnection(url, user, password)) {
            System.out.println("Connected to the database!");
        } catch (SQLException e) {
            System.out.println("Database connection failed: " + e.getMessage());
        }
    }
}
