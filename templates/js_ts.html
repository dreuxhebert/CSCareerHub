Getting Started

To begin coding in JavaScript and TypeScript, you’ll need a few tools:

  • Text Editor or IDE:
    • Visual Studio Code: Ideal for JS and TS development due to its rich ecosystem of extensions.
    • WebStorm: A powerful IDE tailored for JavaScript frameworks.
  • Node.js: Install Node.js to execute JavaScript outside the browser and manage packages with npm (Node Package Manager).
  • TypeScript: Install TypeScript globally using npm: 
    npm install -g typescript

Your First JavaScript Program

Let’s start with a simple "Hello, World!" example in JavaScript:


// JavaScript Hello World
console.log("Hello, World!");

Run this in:

  • The browser console (right-click > Inspect > Console).
  • Node.js: 
    node hello.js

Your First TypeScript Program

Here’s the same "Hello, World!" example written in TypeScript:


// TypeScript Hello World
const message: string = "Hello, World!";
console.log(message);

Compile and Run:

  1. Save the file as hello.ts.
  2. Compile it to JavaScript: 
    tsc hello.ts
  3. Run the compiled file: 
    node hello.js

JavaScript Basics

Variables

JavaScript uses var, let, and const for variable declarations:


// Variables in JavaScript
let name = "Alice";  // Block-scoped
const age = 30;      // Constant
var city = "New York"; // Function-scoped

console.log(`${name} is ${age} years old and lives in ${city}.`);

Control Flow

Conditional statements and loops are used to control the flow of a program:


// Conditional Statement
let num = 10;
if (num > 5) {
    console.log("Greater than 5");
} else {
    console.log("5 or less");
}

// Loop
for (let i = 0; i < 3; i++) {
    console.log(`Iteration: ${i}`);
}

TypeScript Basics

Typing

TypeScript enforces types to make your code more robust:


// Variables with Types
let username: string = "Alice";
let isActive: boolean = true;
let scores: number[] = [95, 88, 76];

console.log(`${username} is active: ${isActive}`);

Interfaces

Use interfaces to define object shapes:


// Interface Example
interface User {
    name: string;
    age: number;
}

const user: User = {
    name: "Alice",
    age: 25,
};

console.log(user);

Object-Oriented Programming

Both JS and TS support object-oriented programming:


// TypeScript Class Example
class Person {
    name: string;

    constructor(name: string) {
        this.name = name;
    }

    greet() {
        return `Hello, ${this.name}!`;
    }
}

const person = new Person("Alice");
console.log(person.greet());

Modules and Imports

Organize your code into reusable modules:


// math.js
export function add(a, b) {
    return a + b;
}

// main.js
import { add } from './math.js';
console.log(add(2, 3));

Final Tips for Beginners

  • Start by mastering JavaScript fundamentals before diving into TypeScript.
  • Use browser developer tools to debug and experiment with JavaScript code.
  • Practice building small projects like a to-do list or a calculator to strengthen your skills.
  • Explore frameworks like React (JS) or Angular (TS) for advanced development.
